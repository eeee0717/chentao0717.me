| 快捷键  | 用途                       |
| ------- | -------------------------- |
| Shift-K | 显示函数文档               |
| gsa     | 包裹代码段(引号或括号)     |
| di"     | 删除引号内的内容(括号同理) |
| zc/zo   | 折叠代码                   |
