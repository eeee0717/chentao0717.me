## 存储引擎概述
存储引擎就是存储数据、建立索引、更新/查询数据等技术的实现方式。

创建表时指定存储引擎
```sql
create table 表名(
    列名1 数据类型1,
    ...
    列名n 数据类型n
) engine=存储引擎名;
```
查看当前数据库支持的存储引擎
```sql
show engines;
```

## InnoDB
InnoDB是Mysql的默认存储引擎，支持**事务**、**行级锁**、**外键**等特性。
> 了解逻辑存储结构

## MyISAM
MyISAM不支持**事务**、**行级锁**、**外键**等特性。

支持**表锁**、**访问速度快**、占用空间小。

## Memory
Memory将数据存储在内存中，只能做缓存表。

特点：内存存放，hash索引。
